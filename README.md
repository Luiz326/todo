## Como baixar o programa

```bash
# Clonar o projeto

$ git clone https://github.com/Luiz326/todo.git

# Entrar no diretório
$ cd todo

# Instalar as dependências
npm install -g @angular/cli 
npm install


# Iniciar o projeto

ng serve -o
```
